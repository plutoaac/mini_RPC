# RPC Server 流程图

本文档详细描述了 RPC 服务器的架构和运行流程。

## 1. 整体架构图

```
┌─────────────────────────────────────────────────────────────────────────┐
│                              RPC Server                                  │
│                                                                          │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────────────────┐  │
│  │              │    │              │    │                          │  │
│  │  RpcServer   │───>│  Connection  │───>│    ServiceRegistry       │  │
│  │              │    │              │    │                          │  │
│  │  - port_     │    │  - fd_       │    │  - handlers_ (map)       │  │
│  │  - registry_ │    │  - registry_ │    │                          │  │
│  │              │    │  - buffers   │    │  "Service.Method"        │  │
│  └──────────────┘    └──────────────┘    │      -> Handler          │  │
│         │                   │            └──────────────────────────┘  │
│         │                   │                      ▲                   │
│         │                   └──────────────────────┘                   │
│         │                                                              │
└─────────│──────────────────────────────────────────────────────────────┘
          │
          ▼
    ┌─────────┐
    │  TCP    │
    │ Socket  │
    └─────────┘
```

## 4. 请求解析流程 (Connection::TryParseRequests)

```
                    ┌─────────────────────────┐
                    │ TryParseRequests() 开始 │
                    │ offset = 0              │
                    └────────────┬────────────┘
                                 │
                                 ▼
              ┌───────────────────────────────────────┐
              │  检查: buffer.size() - offset >= 4 ?  │
              │  (是否有足够字节读取帧头)             │
              └───────────────────┬───────────────────┘
                                  │
                       ┌──────────┴──────────┐
                       │      有帧头?        │
                       └──────────┬──────────┘
                         否 │           │ 是
                            ▼           ▼
                  ┌──────────────┐  ┌───────────────────────────┐
                  │ 移除已处理数据│  │ DecodeFrameHeader()       │
                  │ erase(0,off) │  │ 解析帧头获取 body_length  │
                  │ return true  │  └─────────────┬─────────────┘
                  └──────────────┘                │
                                        ┌────────┴────────┐
                                        │   帧头合法?     │
                                        └────────┬────────┘
                                          否 │         │ 是
                                             ▼         ▼
                                   ┌─────────────┐  ┌────────────────────────────┐
                                   │ return false│  │ 检查: 有完整消息体?        │
                                   │             │  │ buffer.size()-offset >=    │
                                   │             │  │   4 + body_length ?        │
                                   └─────────────┘  └─────────────┬──────────────┘
                                                               │
                                                    ┌──────────┴──────────┐
                                                    │    消息体完整?      │
                                                    └──────────┬──────────┘
                                                      否 │           │ 是
                                                         ▼           ▼
                                           ┌──────────────┐  ┌─────────────────────────┐
                                           │ 等待更多数据 │  │ 提取消息体 (零拷贝)     │
                                           │ break        │  │ body_view = buffer[...] │
                                           └──────────────┘  └───────────┬─────────────┘
                                                                         │
                                                                         ▼
                                                           ┌───────────────────────────────┐
                                                           │ 解析 RpcRequest               │
                                                           │ request.ParseFromArray(...)   │
                                                           └───────────────┬───────────────┘
                                                                           │
                                                                  ┌────────┴────────┐
                                                                  │   解析成功?     │
                                                                  └────────┬────────┘
                                                                    否 │         │ 是
                                                                       ▼         ▼
                                             ┌─────────────────────┐  ┌─────────────────────────┐
                                             │ 生成错误响应        │  │ HandleOneRequest()      │
                                             │ error_code=PARSE_ERR│  │ 处理请求，生成响应      │
                                             │ error_msg="parse..."│  └───────────┬─────────────┘
                                             └──────────┬──────────┘              │
                                                        │                         │
                                                        └────────────┬────────────┘
                                                                     ▼
                                                       ┌───────────────────────────────┐
                                                       │ QueueResponse(response)       │
                                                       │ 将响应加入发送队列            │
                                                       └───────────────┬───────────────┘
                                                                       │
                                                                       ▼
                                                       ┌───────────────────────────────┐
                                                       │ offset += 4 + body_length     │
                                                       │ 移动到下一个帧                │
                                                       └───────────────┬───────────────┘
                                                                       │
                                                                       ▼
                                                       ┌───────────────────────────────┐
                                                       │ 继续处理下一个帧 (循环)       │
                                                       └───────────────────────────────┘
```

## 7. 数据发送流程 (Connection::FlushWrites)

```
                    ┌─────────────────────────┐
                    │ FlushWrites() 开始      │
                    │ sent = 0                │
                    └────────────┬────────────┘
                                 │
                                 ▼
              ┌─────────────────────────────────────┐
              │                                     │
              │       发送循环 (while sent < size)  │
              │                                     │
              └──────────────┬──────────────────────┘
                             │
                             ▼
              ┌─────────────────────────────────────┐
              │ send(fd, buffer + sent,             │
              │      remaining, MSG_NOSIGNAL)       │
              └──────────────┬──────────────────────┘
                             │
              ┌──────────────┴──────────────┐
              │        返回值 rc?           │
              └──────────────┬──────────────┘
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              ▼              ▼
        ┌──────────┐  ┌───────────┐  ┌───────────┐
        │  rc > 0  │  │rc < 0 &&  │  │  其他错误  │
        │ 成功发送 │  │ EINTR     │  │           │
        └────┬─────┘  └─────┬─────┘  └─────┬─────┘
             │              │              │
             ▼              ▼              ▼
    ┌────────────────┐ ┌────────────┐ ┌────────────┐
    │ sent += rc     │ │ continue   │ │ return false│
    │ 继续发送       │ │ 重试       │ │            │
    └───────┬────────┘ └────────────┘ └────────────┘
            │
            │ (全部发送完成)
            ▼
    ┌────────────────────────────┐
    │ erase(0, sent)             │
    │ 清空已发送数据             │
    │ return true                │
    └────────────────────────────┘
```

```

│                                                                        │

```

## 12. 总结

### 核心组件职责

| 组件 | 职责 | 关键方法 |
|------|------|----------|
| **RpcServer** | 服务器生命周期管理，监听端口，接受连接 | `Start()` |
| **Connection** | 单个客户端连接处理，协议解析，请求路由 | `Serve()`, `HandleOneRequest()` |
| **ServiceRegistry** | 服务方法注册与查找，线程安全 | `Register()`, `Find()` |

### 设计特点

1. **阻塞式设计**: 采用同步阻塞 I/O，简单可靠
2. **RAII 资源管理**: `UniqueFd` 自动管理 socket 生命周期
3. **协议简洁**: 4字节长度前缀 + Protobuf 序列化
4. **线程安全注册表**: `ServiceRegistry` 支持并发访问

### 请求处理流程概要

```
客户端连接 -> 读取数据 -> 解析帧 -> 查找方法 -> 执行处理 -> 发送响应 -> 循环
```
